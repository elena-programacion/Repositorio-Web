Símbolos y caracteres especiales

Ejemplos de uso

En este documento, se muestran algunos ejemplos de cómo utilizar símbolos HTML y caracteres especiales en el código HTML.

Símbolos de ángulo (< y >)

Los símbolos de ángulo (< y >) se utilizan para indicar el **inicio** y el **final** de las etiquetas HTML. Por ejemplo: 
<p>Este es un párrafo de texto.<p>

Símbolos de entidad HTML (&)

El símbolo de entidad HTML (&) se utiliza para representar el carácter ampersand (&). Por ejemplo: 
Esta frase contiene un &ampersand.

Comillas

HTML permite utilizar diferentes tipos de comillas para delimitar el texto, como comillas simples (') y comillas dobles (").

Ejemplo con comillas simples: 'Este es un ejemplo de texto entre comillas simples.'

Ejemplo con comillas dobles: "Este es un ejemplo de texto entre comillas dobles."
